Directives

Directives are used to Extend the functionality of the HTML elements.

There are two types of directives in AngularJS.

  • Attribute directives : used for change the behaviour / appearance of the element
  • Structural directives : used for add / remove / modify the HTMl DOM structure
Attribute Directives
  • ngClass : used for adding / removing css classes dynamically in html elements
  • ngStyle : used for adding / removing inline styles dynamically in html elements
Structural Directives
  • *ngIf : used for conditionally displaying the elements
  • *ngFor : used for iterating over the elements
  • *ngSwitch : used for conditionally displaying the elements
